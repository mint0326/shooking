# 장바구니 프로젝트

## 프로젝트 개요

프로젝트 목표와 주요 기능에 대한 간략한 설명
이 프로젝트는 React를 사용하여 패션 쇼핑몰의 장바구니 페이지를 구현하는 것을 목표로 합니다. 주요 기능으로는 사용자가 담은 상품의 수량을 증감시키고, 수량에 따른 상품별 금액 및 총합계 금액을 정확하게 계산하는 것입니다. 또한, 10만원 이상 구매 시 배송비를 무료로 전환하는 핵심 정책을 반영합니다.

참여한 팀원: 정민지

## 개발 환경

사용한 프레임워크, 라이브러리, 언어 등
- Framework: React 18.2.0
- Language: JavaScript (ES6+), JSX
- State Management: React Hooks (Context API, useState, useContext)
- Styling: CSS (또는 Tailwind CSS)
- Build Tool: Create React App

## 설치 및 실행 방법

프로젝트를 로컬에서 실행하기 위한 설치 방법과 명령어

프로젝트 레포지토리를 클론합니다.
git clone <저장소_URL>

프로젝트 디렉토리로 이동합니다.
cd <프로젝트_폴더명>

필요한 패키지를 설치합니다.
npm install

개발 서버를 실행합니다.
npm start

브라우저에서 http://localhost:3000으로 접속하여 확인합니다.

## 컴포넌트 구조

주요 컴포넌트들의 구조와 역할

- CartPage: 장바구니 페이지 전체를 감싸는 최상위 컴포넌트입니다. 헤더, 상품 목록, 배송비 안내, 주문 요약 컴포넌트를 포함합니다.
- Header: (기존 Header 재사용) '상품 목록'으로 돌아갈 수 있는 뒤로가기 버튼 또는 '장바구니 탭' 기능을 제공합니다.
- CartItem: 장바구니에 담긴 개별 상품을 표시합니다. 수량 증감(+, -) 버튼이 있으며, 버튼 클릭 시 실시간으로 상품 1개 가격과 수량에 따른 총가격을 모두 다시 계산하여 보여줍니다.
- ShippingInfoBanner: '10만원 이상 구매 시 무료배송' 정책을 명확히 명시하는 고정 배너 컴포넌트입니다.
- OrderSummary: 페이지 최하단에 위치하는 주문 요약 컴포넌트입니다. CartItem들로부터 계산된 총 선택한 제품 가격을 받아오고, 이 가격을 기준으로 **배송료(3,000원 또는 0원)**를 계산하여 최종 결제 금액을 실시간으로 보여줍니다.
- CheckoutButton: (결제 모듈 연동) '결제하기' 버튼을 클릭하면, 이전에 개발한 결제 모듈(모달)을 활성화시킵니다.

## 테스트 방법

테스트 케이스 및 실행 방법

자동화된 단위/컴포넌트 테스트를 실행합니다.
npm test

본 프로젝트는 CartItem, OrderSummary, ShippingInfoBanner 등 재사용 가능한 핵심 컴포넌트를 Storybook으로 문서화하여 시각적 테스트를 진행합니다.

기능 테스트 작업 (필수 확인)
작업한 페이지 URL에 접속하여, 아래 기능들이 동작되는지 확인.
- 장바구니에 담긴 상품들의 금액이 정확하게 계산되서 보이나요? (수량 변경 시 실시간 반영)
- 10만원 미만인 경우 배송비 3,000원, 10만원 이상인 경우 배송비가 무료로 보이나요?
- 뒤로가기 버튼이나 상단 탭을 이용해 메인 상품 목록 페이지로 이동할 수 있나요?
- 결제하기 버튼을 누르면, 결제 할 수 있는 창이 보이나요? 그리고 카드 등록 및 결제가 가능한가요?

## 유의사항 및 알려진 이슈

배송료 정책은 현재 '10만원 이상' 무료 배송 기준으로 고정되어 있습니다. (예: 도서산간 추가 배송비 미적용)
