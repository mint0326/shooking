# 장바구니 페이지 리뷰

## 고객사 요구사항
장바구니에 담은 물건의 수량을 더하거나 뺄 수 있어야 합니다.
수량을 바꾸면 금액이 바로바로 다시 계산되어야 합니다.
상품 1개 가격과, 수량만큼 곱한 총가격이 모두 보여야 합니다.
페이지 맨 아래에 (총 상품 금액 + 배송비 = 최종 결제 금액)이 정확하게 나와야 합니다.
"10만원 이상 구매 시 무료배송" 문구가 눈에 잘 띄게 보여야 합니다.
총 상품 금액이 10만원이 넘으면, 배송비가 자동으로 "0원"이 되어야 합니다.

## 요구사항에 따른 기능 동작 확인

1. 수량 변경 및 금액 자동 계산
장바구니에 담긴 상품 옆에 있는 + / - 버튼이 잘 작동합니다.
+버튼을 누르면 수량이 2개, 3개로 늘어나고, 동시에 수량에 맞게 상품 가격이 자동으로 다시 계산됩니다. (예: 30,000원짜리 상품 3개 = 90,000원)

2. 10만원 이상 무료배송 (배송비 자동 변경)
총 상품 금액이 10만원이 안 될 때는 최종 결제 금액에 "배송비 3,000원"이 더해집니다.
물건을 더 담아서 총금액이 10만원을 넘어가면, 즉시 '배송비 0원'으로 바뀌면서 배송비가 사라집니다.

3. 정확한 최종 결제 금액
페이지 맨 아래에 "총 상품 금액"과 "배송비"가 따로 표시되고, 두 금액을 합친 "최종 결제 금액"이 정확하게 나오는 것을 확인했습니다. 수량을 바꿀 때마다 이 최종 금액도 실시간으로 바뀝니다.

4. 안내 문구 표시
고객이 바로 알 수 있도록 "10만원 이상 구매 시 무료배송" 안내 문구를 페이지에 잘 보이도록 추가해 두었습니다.
