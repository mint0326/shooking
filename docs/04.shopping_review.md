# 쇼핑몰 연동 리뷰

## 고객사 요구사항
[핵심] 자연스러운 페이지 이동: 기존에 따로따로 확인했던 '상품 목록', '장바구니', '결제' 화면이 끊김 없이 자연스럽게 연결되어야 합니다.
[신규] 상세 페이지 제작: 상품 목록에서 사진을 눌렀을 때, 더 큰 사진과 상세 설명, 가격을 볼 수 있는 '상세 페이지'를 새로 만들어야 합니다.
[중요] 장바구니 숫자 유지: 상세 페이지에서 물건을 담든, 목록에서 담든 상단에 있는 '장바구니 아이콘 숫자'가 실시간으로 똑같이 반영되어야 합니다. 페이지를 왔다 갔다 해도 담아둔 물건이 사라지면 안 됩니다.
[디자인] 시안 반영: 이메일로 보내주신 디자인 시안처럼 상세 페이지 하단에 **'관련 상품'**이 추천되도록 배치해야 합니다.
[확인] 테스트 주소 제공: 작업한 내용을 저희(고객사)가 스마트폰으로 직접 눌러볼 수 있는 인터넷 주소를 전달해 주세요.

## 요구사항에 따른 기능 동작 확인
- 자연스러운 화면 이동
확인 결과: 상품 목록에서 마음에 드는 신발을 클릭하면 상세 페이지로 부드럽게 넘어갑니다. 상세 페이지에서 '장바구니 담기'를 누른 뒤, 우측 상단 장바구니 아이콘을 누르면 즉시 장바구니 화면으로 이동합니다. 앱을 쓰는 것처럼 자연스럽게 연결해 두었습니다.

- 상세 페이지 및 디자인 구현
확인 결과: 보내주신 이미지 시안 그대로 구현했습니다.
상단에는 상품 이미지가 크게 보이고, 그 아래로 상품명과 가격이 잘 정돈되어 보입니다.
화면 맨 아래쪽에는 요청하신 대로 '함께 보면 좋은 상품' 목록이 가로로 나열되어 있어, 고객이 다른 상품도 쉽게 구경할 수 있습니다.

- 장바구니 정보 실시간 동기화 (가장 중요한 기능)
확인 결과: 상세 페이지에서 수량을 '2개'로 설정하고 장바구니에 담으면, 화면 맨 위 장바구니 아이콘 숫자가 바로 '2'로 바뀝니다. 다시 '뒤로가기' 버튼을 눌러서 메인 화면으로 돌아와도 숫자는 여전히 '2'로 유지됩니다. 어떤 화면에 있더라도 고객이 담은 물건의 개수를 정확하게 기억하고 보여줍니다.
