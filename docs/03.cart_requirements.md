# 장바구니 페이지 요구사항 분석 및 기능 목록

## 📌 기본 정보
### 프로젝트명: 
패션 쇼핑몰 장바구니 페이지

### 사용하게 될 기술: 
React

### 문서 작성자: 
정민지

## 📝 고객 요구사항 정리
- 장바구니 페이지
  - 금액의 정확성 (수량 및 할인, 가격, 배송료 모두 반영하여 가격 산출)
  - 10만원 이상 구매시 무료배송 '꼭' 명시 해두기.
- FAST URL 제공

## 📋 기능 목록
- 제품 수량 증감 기능
- 제품별 가격 (1개 가격과 수량만큼 곱한 가격 모두 보이게)
- 제일 하단에 총 선택한 제품 가격 + 배송료 가격
- 10만원 이상 구매시 배송비 무료로 전환
- 장바구니로 이동할 수 있는 버튼이나 탭 생성.
